# Koda for VS Code

A Visual Studio Code port of the beautiful
[Koda.nvim](https://github.com/oskarnurm/koda.nvim)
theme.

## Features

- Dark minimal UI
- Koda.nvim inspired syntax colors
- Semantic highlighting support
- VS Code workbench customization
